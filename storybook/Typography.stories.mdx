# Typography
제목, 본문을 포함한 기본 텍스트 입니다.

## Headings
타이틀을 `<h1> ~ <h6>` HTML 태그 또는 `.h1 ~ .h6` 클래스 스타일로 지정해 가능합니다.
- `h1`: 44px
- `h2`: 32px
- `h3`: 22px
- `h4`: 18px
- `h5`: 16px
- `h6`: 14px

## Weight
4가지 타입의 텍스트 가중치를 제공합니다.
- `.font-weight-bold`: Bold (700)
- `.font-weight-medium`: Medium (500)
- `.font-weight-normal`: Regular (400)
- `.font-weight-light`: Light (300)
